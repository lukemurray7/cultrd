# Funnel Sans Font Files

This directory should contain the Funnel Sans font files.

## Download Instructions

1. Visit the Funnel Sans GitHub repository: https://github.com/Dicotype/Funnel
2. Download the font files (TTF format)
3. Place the following font files in this directory:
   - `FunnelSans-Regular.ttf`
   - `FunnelSans-Medium.ttf`
   - `FunnelSans-SemiBold.ttf`
   - `FunnelSans-Bold.ttf`

## Font Weights

The project uses these specific weights:
- Regular (400)
- Medium (500)
- SemiBold (600)
- Bold (700)

Make sure the font file names match exactly as listed above for the app to load them correctly.
